## Datasets

Download and unzip the <model name> model benchmarking data.

```
wget https://zenodo.org/record/2531868/files/gnmt_inference_data.zip
unzip gnmt_inference_data.zip
export DATASET_DIR=$(pwd)/nmt/data
```

Set the `DATASET_DIR` to point as instructed above  when running <model name>.
